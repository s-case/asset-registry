S-CASE Asset Registry REST API

Description of S-CASE Asset Registry REST API

Summary

Path Method Description
/artefact POST Creates and stores a new artefact in the repository
/artefact/directsearch GET Finds a list of artefacts by search query
/artefact/exists/{id} GET Checks whether an artefact exists in the repository
/artefact/search GET Finds a list of artefacts by free text search strings
/artefact/{id} DELETE Deletes an artefact from the repository
GET Finds an artefact in the repository by ID
PUT Updates an artefact in the repository
/domain/domains GET Retrieves a list of all domains in the repository
/domain/{id} GET Finds a domain by ID in the repository
/project POST Creates and stores a new project in the repository
/project/directsearch GET Finds a list of projects in the repository by search query
/project/search GET Finds a list of projects by free text search strings
/project/{id} DELETE Deletes a project from the Asset Repo by ID
PUT Updates a project in the Asset Repo
/project/{id}/id GET Finds a project in the repository by ID
/project/{name} DELETE Deletes a project from the Asset Repo by name
/project/{name}/name GET Finds a project in the repository by its name
/subdomain/subdomains GET Retrieves a list of all subdomains in the Asset Repository
/subdomain/{id} GET Finds a subdomain by ID in the Asset Repository
/version GET Returns the version number
